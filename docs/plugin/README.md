# Plugin Docs

This section documents the cloud plugin system.

## Files

- `overview.md`: high-level architecture and plugin discovery.
- `plugin-yml.md`: `redstonecloud.yml` / `plugin.yml` schema.
- `lifecycle.md`: load/enable/disable order and dependency resolution.
- `classloading.md`: class loading behavior and resource access.
- `commands.md`: command registration and completion trees.
- `listeners.md`: event subscriptions and priorities.
- `tasks.md`: scheduler usage and task lifecycle.
