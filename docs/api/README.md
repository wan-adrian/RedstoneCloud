# API Module Docs

This section documents the `api` Maven module (`de.redstonecloud:api`).

## Files

- `overview.md`: module responsibilities and package map.
- `broker-and-packets.md`: Redis broker, message model, packet registry, and default packet set.
- `cache-and-crypto.md`: cache wrappers, key utilities, and RSA helpers.
- `integration.md`: practical integration flow and caveats.
- `custom-packets.md`: generic guide for implementing and operating custom packets.
