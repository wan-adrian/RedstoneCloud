# RedstoneCloud Docs

Documentation is organized by domain.

## Index

- `docs/cloud/`: cloud-only installation, bootstrap flow, and configuration.
- `docs/rest-api/`: optional cloud REST API (config, auth, endpoints, examples).
- `docs/api/`: Java `api` module architecture (components, broker, packets, cache, crypto), including custom packet integration notes.

## Cloud-only Note

This documentation set is focused on **single cloud instance mode** (no node setup).
Cluster/node docs are intentionally out of scope for now.
